Python数组

为什么要用数组

列表（list）既灵活又简单，但面对各种需求的时候，比如，存放1000万个浮点数的话，数组（array）的效率就比列表高很多。因为数组背后存放的并不是float对象，而是数字的机器翻译，也就是字节表述。就跟C语言的数组一样。
